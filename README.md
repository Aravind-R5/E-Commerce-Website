# NexGenGear — E-Commerce Gaming Store

A modern full-stack **E-Commerce Platform** built with Django and React, specializing in premium gaming hardware.

## ⚡ Tech Stack

| Layer | Technology |
| :--- | :--- |
| **Backend** | Python 3.12, Django, Django REST Framework |
| **Auth** | JWT (SimpleJWT) |
| **Database** | PostgreSQL (Production) / SQLite (Local) |
| **Frontend** | React (Vite) |
| **Styling** | Custom Vanilla CSS (Design System) |
| **HTTP** | Axios with interceptors |

## 🎯 Features

🔶 **Customer** — Browse products by category, view detailed product pages, add to cart, seamless checkout flow, manage order history in personal dashboard.

🔶 **Store Admin** — Comprehensive admin dashboard to view total sales and orders, manage customer orders, update order statuses (Pending, Processing, Shipped, Delivered), and manage product inventory.

🔶 **UI** — Premium aesthetic with custom color palettes, responsive mobile-first layout, beautiful product grid displays, and dynamic hover animations.
